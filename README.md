# Qrar Landing Page

Marketing site for Qrar (qrar.ai) — bilingual (EN/AR) React + Vite + Tailwind.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:8080

## Production build

```bash
npm run build
npm run preview
```

Static output is written to `dist/`. Deploy to any static host (Vercel, Netlify, Cloudflare Pages, S3 + CloudFront, Nginx).

## Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS + shadcn/ui
- React Router
